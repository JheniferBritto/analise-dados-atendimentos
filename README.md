# Análise de Dados de Atendimentos

## Objetivo
Analisar dados de atendimentos para identificar problemas, tratar inconsistências e gerar insights.

## Processo
- Exploração dos dados
- Tratamento de valores nulos
- Padronização de colunas
- Análise de outliers
- Criação de métricas

## Principais decisões
- Remoção da coluna Data Atendimento (dados inconsistentes)
- Tratamento da coluna Quantidade com mediana
- Padronização da coluna Cidade

## Insights
- Identificação de valores fora da curva
- Diferença de valor por atendimento entre cidades e especialidades
